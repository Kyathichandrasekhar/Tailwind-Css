# Tailwind CSS Practice

This repository contains my Tailwind CSS learning and practice code. It includes examples of commonly used Tailwind utility classes and layouts to understand how Tailwind works.

## Topics Covered

- Installation of Tailwind CSS
- Background Colors
- Text Styling
- Flexbox
- Grid Layout
- Width & Height Utilities
- Spacing (Margin & Padding)
- Borders & Border Radius
- Colors
- Responsive Design Basics
- Hover Effects
- Buttons and Cards

## Technologies Used

- HTML5
- Tailwind CSS
- Node.js
- npm

## Purpose

The purpose of this repository is to practice and understand Tailwind CSS concepts through hands-on examples.

## Project Structure

```
Tailwind-css/
│── dist/

│── package.json

│── README.md
```

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Install dependencies

```bash
npm install
```

3. Start Tailwind

```bash
npm run dev
```
